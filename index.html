Hubway Challenge

Project URL: https://github.com/dzwarg/hubway-challenge

Description: The Hubway Visualization Challenge was a contest run in Boston during the Fall of 2012 to visualize 500,000+ bike trips on the Hubway system. My entry into the contest was twofold:

  • d3 visualizations: Using d3, I had a web app look at cross sections of the rider populations in three different ways. Each of the visualization cross sections linked to the other sections, so you could traverse the visualization modes seamlessly.
  • graphserver visualizations: Using graphserver, I used the routing engine in graphserver to composite trips for segments of time together, and visualize where people were traveling, based on their starting point and ending point. This final result was compiled into a movie. The final compilation is on youtube.

SpamLibs

Project URL: https://github.com/dzwarg/spamlibs
App URL: http://spamlibs.appspot.com

Description: SpamLibs takes spam email messages, and uses them for Mad Libs™. The application runs the body of the spam email through a natural language processor to determine the types of words in the message (noun, adjective, verb). The application then randomly removes many of the original words, and prompts users to fill in the missing types of speech. After a user has submitted the form, the application takes the user submitted vocabulary and combines it with the original spam email, often to hilarious effect!

vTour

Project URL: https://github.com/dzwarg/vtour
App URL: http://www.spondubone.net/~davidz/vtour_app

Description: vTour is a method of searching for geotagged photographs on 3 major photo sharing APIs:

  • Flickr
  • Panoramio
  • Picasa
A user can search by photos by selecting a place on the map, which brings up all photos that are geolocated within a certain distance of that place. The app performs a bounding-box search, since not all APIs support radius searching.

This app uses the python app FeatureServer to save search results, which in turn saves features into a PostGIS database.
